# betternpx

Reserved package name for Better npm's safer `npx` command.

For now, prefer the installer script instead of installing this package directly from npm:

```sh
curl -fsSL https://betternpm.org/latest | sh
```

The installer links the current Better npm CLI aliases: `betternpm`, `bnpm`, `betternpx`, and `bnpx`.

This package reserves the public command name while the product is still being hardened. It intentionally points users to the installer until the scoped CLI package is ready to publish.
