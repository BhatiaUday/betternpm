# Better npm Core

Core package inspector and risk-scoring engine for Better npm.

This package is published to support the Better npm CLI. End users should install the CLI through the installer script while the project is still early:

```sh
curl -fsSL https://betternpm.org/latest | sh
```

The public product is the CLI. `betternpx`/`bnpx` are the `npx` replacements, and `betternpm`/`bnpm` are the `npm` replacements. This package is the shared inspector behind those surfaces.
